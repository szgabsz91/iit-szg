This application is a collection of courses of various topics. Each course contains a set of labs. The labs are recommended to be completed in order.

Currently there are two courses:

  • XML: contains topics such as XDM, DTD, XML Schema, XML related Java and C# APIs, JSON, RSS, XHTML, XPath, XSLT, XQuery and XML DB.
  • Databases: contains topics such as LDAP, JPA, LINQ, ORDBMS, MongoDB, Neo4j and ontologies.

Each course starts with a Prerequisites virtual lab that introduces the necessary applications and how to set them up. The labs after that contain exercises for the given topics, as well as a solution project that can be used for verification purposes. However, it is recommended to solve the exercises without taking a look at the solution.
